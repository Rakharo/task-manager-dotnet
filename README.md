Projeto simples de Task Managaer para entendimento e aplicação de conceitos de backend C# .NET;

Para aprendizagem, o projeto foi arquitetado em Controller - Service - DTO - Repository - Model;

Com conexão com banco de dados simples (utilizando HeidiSQL) e Entity Framework;

O objetivo, como dito antes, era aplicar os conceitos de estrutura backend para uma webapi utilizando c# e dotnet.

Ao rodar o projeto (dotnet run) é possivel rodar localhost a url que leva ao Swagger (http://localhost:5209/swagger), para testar os endpoints criados.
